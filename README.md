# trpc-postgres-react-daisyui-app

A React app without GraphQL

# Stacks

**General**

- Node.js
- TypeScript
- pnpm
- esbuild
- tRPC

**Server**

- Kysely
- Postgres

**Frontend**

- Vite
- React
- Tailwind CSS
- Daisy UI

No GraphQL.

# Getting Started

```
make dev
```
