# trpc-postgres-react-daisyui-app

A React app without GraphQL

# Stacks

**General**

- Node.js
- TypeScript
- pnpm
- esbuild
- tRPC

**Server**

- Kysely
- Postgres

**Frontend**

- Vite
- React
- Tailwind CSS
- Daisy UI

No GraphQL.

# Getting Started

```
docker-compose up -d

cd ./frontend
pnpm install
cd ..

cd ./server
pnpm install
cd ..

make dev
```
