# fastify-daisyui-serverside-app

An example of traditional CRUD app without a Frontend framework. Just HTML + CSS.

# Stacks

**General**

- Node.js
- TypeScript
- pnpm
- esbuild

**Server**

- Fastify
- Kysely
- SQLite (for demo purpose)

**Frontend**

- react-dom/server (as a template)
- Tailwind CSS
- Daisy UI

No React.

# Getting Started

```
pnpm install
pnpm dev
```
